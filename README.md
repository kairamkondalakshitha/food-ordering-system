# 🍔 Food Ordering System

A full-stack Food Ordering System developed as an internship project.

Users can browse restaurants, add food items to cart, view orders, and manage their account.

---

## 🚀 Features

### Frontend
- Responsive food ordering interface
- Home page
- Restaurant listing
- Shopping cart
- Order history
- Login page
- Registration page

### Backend
- Node.js and Express server
- MongoDB database connection
- REST API structure
- User authentication setup

---

## 🛠️ Technologies Used

### Frontend
- React.js
- React Router
- Bootstrap
- HTML
- CSS
- JavaScript

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
