# Online Food Ordering & Restaurant Management System

## Project Description
This project is a Full Stack Online Food Ordering & Restaurant Management System developed using Node.js, Express.js, MongoDB, JWT Authentication, and REST APIs.

## Features

### Customer
- Register/Login
- Browse Restaurants
- Search Food Items
- Add to Cart
- Place Orders
- Order Tracking
- Order History
- Ratings & Reviews

### Restaurant Owner
- Manage Restaurants
- Add Food Items
- Manage Orders

### Delivery Partner
- View Assigned Orders
- Update Delivery Status

### Admin
- Manage Users
- Manage Restaurants

## Technologies Used
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- REST APIs
- Morgan
- CORS
- Docker

## API Endpoints

- POST `/api/auth/register`
- POST `/api/auth/login`
- GET `/api/restaurants`
- POST `/api/restaurants`
- GET `/api/foods`
- POST `/api/foods`
- GET `/api/cart`
- POST `/api/cart`
- GET `/api/orders`
- POST `/api/orders`
- PUT `/api/orders/:id/status`
- GET `/api/reviews`
- POST `/api/reviews`
